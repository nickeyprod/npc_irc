<h2><b>База данных</b></h2><br>
Данная модель БД описывает две сущности: Вакансия (vacancy) и кандидат (сandidate).<br>
Вакансия и кандидат связаны внешним ключом по полю *for_vacancy_id* кандидата, связью 1 ко многим.<br>
Много кандидатов на одну вакансию, остальные поля описаны в Физической модели БД на схеме ниже.<br>

<p align="center">
<img src="d2.schema.png" alt="Физическая модель БД">
</p>

Данная БД создается и заполняется произвольными данными файлом, лежащим в корне проекта, под названием init-db.sql.<br>
<br>
Чтобы создать новую БД в соответствии с физической моделью выше, запустите команду в терминале: <br>
```bash
psql -U postgres -d postgres -f /путь/к/файлу/init-db.sql
```
Если вы находитесь в корне проекта, запустите команду:<br>
```bash
psql -U postgres -d postgres -f ./init-db.sql
```
Чтобы подключиться к только что созданной БД, запустите команду:<br>
```bash
psql -U postgres -d npc_irc_db
```
<br>
После успешного запуска скрипта и инициализации БД, можно билдить проект и запускать сервер Node.js.<br><br>
<b>P.S: Проект уже скомпилирован и собран. Можно сразу переходить к пункту 3 (Запуск проекта), но если что-то не работает, можно скомпилировать / сбилдить его самому, пройдя через пункты 1,2.</b>
<br>
<h2><b>1. Клиентская часть</b></h2><br>

Для клиенткой части был взят бесплатный темплейт для React **Argon Dashboard Material UI**. Исходный код взят с официального сайта https://www.creative-tim.com из связанного репозитория на github.com. <br> 
Проект уже собран, файлы React находятся в папке distr/client. Если требуется собрать клиентскую часть React заново, необходимо иметь установленный пакетный менеджер npm.<br> Переидите с помощью терминала в папку distr/client и запустите сначала команду 
```bash
npm install
```
<br>
После чего запустите команду:<br>

```bash
npm run build
```
<br> Файлы клиенткой части появятся в папке distr/client/build<br>
<h2><b>2. Серверная часть</b></h2><br>
Серверная часть написана на языке Typescript, скомпилированные файлы JS находятся в папке distr/server. Если требуется скомпилировать файлы заново, необходимо иметь установленный Typescript Compiler (tsc), а также установленный пакетный менеджер npm.<br> Переидите с помощью терминала корневой каталог NPC_IRC и запустите сначала команду 
```bash
npm install
```
<br>
После чего запустите команду:<br>

```bash
tsc
```
После вышеизложенных процедур, JS файлы серверной части появятся в папке /distr/server.<br>
<h2><b>3. Запуск проекта</b></h2><br>
Теперь когда все файлы скомпилированы/собраны, переидите с помощью терминала корневой каталог NPC_IRC и запустите проект командой: <br>

```bash
npm run start
```

Если вы увидели в терминале надпись *[NPC IRS] App is now running at http://localhost:5003*, значит проект успешно запустился!<br>
Откройте браузер и введите в адресную строку указанный адрес: *http://localhost:5003/*, вы должны увидеть Dashboard и боковое меню слева, в котором первые два пункта рабочие ( первый это сам Dashboard, второй пункт Таблицы), в котором находится выполненное задание.
